# Explog

## Explog 소개

- #### 안드로이드 스튜디오를 이용해 여행 어플리케이션인 Explog(Explore + Log)를 개발하였다.
- #### 앱의 전체적인 기능은 여행앱인 '여행노트'와 'VOLO'를 참고하였고, UI는 카카오플레이스를 참고하였다.
